## 2. 프로젝트 계약 (설계 전제)

| 항목 | 내용 |
|------|------|
| 입력 | 4×4 `int[][]`, `0`=빈칸(정확히 2개), 값 `0` 또는 `1~16`, non-zero 중복 금지 |
| 출력 | `int[6]` = `[r1,c1,n1,r2,c2,n2]`, 좌표 **1-index** |
| Magic Constant | M = **34** (`MagicConstant` SSOT) |
| 검증 순서 (short-circuit) | null → size → empty count → value range → duplicate |
| Boundary invalid | Failure envelope 반환 (**Python 예외 아님**) |
| E00x ↔ PRD | `E003` null · `E001` size · `E002` empty · `E004` range · `E005` duplicate |
| invalid 시 Domain | `SolvePartialMagicSquare.execute` / `SolveTwoBlanksUseCase.execute` **0회** (U-FLOW-02) |
